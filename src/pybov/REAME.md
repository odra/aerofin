# PyBOV

Ibovespa python library used in the aerofin project.

Further details on B3 data format can be found in their website: https://www.b3.com.br/en_us/market-data-and-indices/data-services/market-data/historical-data/equities/historical-quote-data/

## Features

* API and Data models to parse  B3's historical archives
* A CLI to parse data from both files and STDIN

## Development

This project uses `mypy` in strict mode for code validation and `poetry` project tooling.

A `Makefile` is provided for convenience.
